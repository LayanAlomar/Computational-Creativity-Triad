# Computational Creativity Triad (CCT)

## Overview
This repository contains the research paper introducing the Computational Creativity Triad (CCT), a framework for evaluating creativity in AI-generated visual art.

## Research Focus
- AI-Generated Art Evaluation
- Computational Creativity
- Novelty
- Value
- Surprise
- Diffusion Models
- CLIP
- DINOv2
- LAION-Aesthetics

## Highlights
- Introduces the Computational Creativity Triad (CCT)
- Evaluates AI-generated artwork using three creativity dimensions
- Compares CCT against FID, Inception Score, CLIPScore, and LAION-Aesthetics
- Includes experimental results and statistical analysis
